Installation
==============

### CocoaPods

1. Add `pod 'MegaScroller'` to your Podfile.
2. Run `pod install` or `pod update`.
3. Import "MegaScroller.h".

Requirements
==============
This library requires `iOS 6.0+` and `Xcode 6.0+`.
